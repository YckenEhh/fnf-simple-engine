# Friday Night Funkin' Simple Engine

Advenced version of original FNF with lot of cool stuff like:
- Auto update system
- Better input
- Botplay
- Custom keybinds
- Downscroll
- Extra key support (1-9 keys)
- Replay system (In beta)
- Note types support
- Better charter
- Middle scroll
- Lane alpha
- And a lot of another stuff!
